# MVP Roadmap

## Phase 1: Concierge MVP

Goal: prove that structured AI onboarding creates better matches before
building a large marketplace.

Build:

- Basic account and dating profile.
- AI onboarding chat.
- Structured profile extraction.
- Admin view for reviewing profiles.
- Manual or semi-automated matching.
- Match explanation shown to both users.
- Mutual approval before chat.
- Post-match feedback.

Avoid:

- Fully autonomous agent-to-agent dating.
- Swipe feed.
- Complex ML ranking.
- Public profile browsing.
- Large social features.

## Phase 2: Semi-Automated Matching

Build:

- Hard filters and compatibility scoring.
- Candidate generation.
- Weekly curated match batches.
- Embedding-based similarity.
- Safer conversation coach.
- Better reporting, blocking, and moderation.

## Phase 3: Learning System

Build:

- Ranking model from accepts, rejections, conversations, and dates.
- Personalized onboarding follow-ups.
- Match quality analytics.
- Niche/community-specific match rules.
- Trust and verification scoring.

## MVP Success Metrics

| Metric | Why It Matters |
| --- | --- |
| Onboarding completion rate | Measures whether users tolerate the agent interview |
| Structured profile completeness | Measures extraction quality |
| Match accept rate | Measures recommendation relevance |
| Mutual accept rate | Measures two-sided fit |
| First-message rate | Measures match confidence |
| Date intent / date completed | Measures real-world value |
| Feedback response rate | Measures learning-loop health |

## Recommended First Niche

Start with one high-intent community instead of a generic dating app. Good
options include Indian professionals in one city, diaspora Indians, serious
relationship seekers, or a women-first WhatsApp-based dating experience.
